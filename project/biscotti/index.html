Biscotti: A Ledger for P2P Machine Learning

 

Can federated learning be made more secure and private?

We leverage blockchain primitives to enable secure peer-to-peer federated learning. A VRF committee verifies and aggregates SGD updates and appends the sum as a block in a ledger.
